First-Order Step Fit (Ka, τ)

This project fits a first-order process model:

y(t) = y0 + Ka(1 - exp(-t / τ))

Run locally

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run gui_app.py

About

Built for a process control lab to estimate Ka and τ from step-response data.
